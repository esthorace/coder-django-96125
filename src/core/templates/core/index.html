Django

The Web framework for perfectionists with deadlines

Django es un framework web de alto nivel, basado en Python, que fomenta el desarrollo rápido y un diseño limpio y pragmático.

Fue construido por desarrolladores experimentados para encargarse de gran parte de la configuración del desarrollo web, permitiéndote concentrarte en escribir tu aplicación sin necesidad de reinventar la rueda.
